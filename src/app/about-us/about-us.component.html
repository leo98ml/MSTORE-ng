About Us... Ehm, Me!




Hi, I'm Leonardo Marafioti and this is my little-ugly-project for my "Piattaforme Software Per Il Web" exam @ UNICAL.

It's a very simple e-commerce that provide the possibility to create an account, login, add items to the shopping cart and actually buy them!

The aim behind all this is to proof that I know a little bit of web developing with enterprise technology such as Java-EE, Spring, Angular and MySQL.

Check the source code for both the front-end and the back-end tiers on my github repo linked under this brief description.
